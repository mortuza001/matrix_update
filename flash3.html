Firmware Flasher

Install the latest firmware directly to your Infomatrix from your browser.

How to use:
  1. Connect your device via USB.
  2. Click Install below.
  3. Select the correct COM port from the list.
Install Firmware
⚠️ Your browser does not support Web Serial.
Please use Google Chrome or Microsoft Edge on a desktop.

Ensure your board is in bootloader mode if it doesn't connect automatically.
